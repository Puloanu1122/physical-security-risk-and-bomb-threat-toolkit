# Physical Security Risk and Bomb Threat Toolkit

## Overview
This repository provides practical guidance, checklists, and frameworks for physical security risk assessment and bomb threat preparedness. It is designed for security professionals responsible for protecting facilities, people, and operations.

## Objectives
- Identify and assess physical security threats
- Support structured risk-based decision making
- Improve bomb threat preparedness and response
- Align security practices with recognized international standards

## Standards & References
- ISO 31000 – Risk Management
- ISO 22301 – Business Continuity
- FEMA / DHS Independent Study guidance
- ASIS Physical Security and GSRA principles

## Repository Structure
- Risk Assessment
- Bomb Threat Management
- Explosive Sweep Guidelines
- Standard Operating Procedures
- Templates

## Intended Audience
- Security Officers
- Assistant Security Managers
- Facility and Operations Managers

## Disclaimer
This material is for training and preparedness purposes only. It does not replace law enforcement, emergency services, or professional security advice.
